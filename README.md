# Grama Bazaar Dashboard

## Project Overview

Grama Bazaar is an agricultural market analytics dashboard designed to help farmers identify the most profitable markets for selling crops. The project integrates market prices, crop information, village locations, and transportation distances to support data-driven decision-making.

## Problem Statement

Farmers often struggle to determine which market offers the best returns for their produce. Market prices vary across locations, and transportation costs can significantly impact profit.

This dashboard addresses the problem by analyzing market profitability and presenting insights through interactive visualizations.

## Objectives

* Identify the most profitable agricultural markets.
* Compare crop prices across different markets.
* Analyze market profitability based on distance and price variations.
* Enable location-based filtering using District, Taluk, Village, and Crop.
