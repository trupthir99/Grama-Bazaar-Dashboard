# Grama Bazaar Dashboard

## Project Overview
Grama Bazaar is an agricultural market analytics dashboard developed using BigQuery, SQL, and Looker Studio.

### Features
- Market Profit Heatmap
- Top Profitable Markets
- Crop-wise Analysis
- District, Taluk, Village Filters
- KPI Cards
- Market Price Comparison

- Identified profitable markets for farmers.
- Compared crop prices across regions.
- Enabled data-driven market selection.
- Visualized market profitability using interactive charts.
